# Event Horizon — Event Management Web App

Front-end web app to manage upcoming events. Built with HTML, CSS, and vanilla JavaScript.

## Features
- Add, delete, and display events dynamically
- Form validation with warning message on incomplete input
- Past events visually distinguished (gray background)
- Auto-sort by date ascending after adding
- Search filter by event name or date
- Responsive layout (desktop, tablet, mobile)

## Tech Stack
- HTML5
- CSS3 (Flexbox / Grid)
- Vanilla JavaScript (DOM manipulation, no frameworks)
